# Product Overview

## AI Agent Framework

A comprehensive platform that enables developers to create, orchestrate, and deploy AI agents with minimal complexity while maintaining enterprise-grade capabilities.

## Core Value Proposition

- **Simplicity First**: Complex AI capabilities exposed through intuitive VS Code-style interfaces
- **Enterprise Ready**: Built-in RBAC, audit trails, guardrails, and compliance features
- **Self-Hosting**: Complete data sovereignty with downloadable deployment packages
- **Visual Orchestration**: BPMN-based workflow design with AI-enhanced automation
- **Extensible**: Plugin architecture for custom tools and MCP server integrations

## Key Features

- **Agent Templates**: Pre-built frameworks for chatbots, content generation, and data analysis
- **Memory System**: Semantic memory with conversation continuity and intelligent management
- **Security by Design**: Multi-layered security with content filtering and access controls
- **LLM Flexibility**: Support for multiple providers (OpenAI, Anthropic, local models)
- **Deployment Options**: From Docker Compose to Kubernetes to cloud-native

## Target Users

- Semi-technical developers building AI-powered applications
- Enterprise teams requiring secure, compliant AI solutions
- Organizations needing self-hosted AI infrastructure with data sovereignty
